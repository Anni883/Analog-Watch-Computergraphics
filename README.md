# ⌚ Interactive 3D Analog Watch

A real-time interactive 3D Analog Watch simulation built using **Three.js**, **WebGL**, **GLSL Shaders**, and **JavaScript** for Computer Graphics coursework.

## 📌 Project Overview

This project demonstrates the implementation of a realistic analog wristwatch in a 3D environment using modern computer graphics techniques. The watch displays the current system time in real-time while providing interactive camera controls, dynamic lighting, custom shaders, procedural textures, and animation effects.

## ✨ Features

* Real-time analog clock synchronized with system time
* Interactive 3D camera controls
* Dynamic lighting system with multiple light color modes
* Mouse-controlled light positioning
* Custom GLSL vertex and fragment shaders
* Procedural texture generation using HTML5 Canvas
* Realistic watch materials and reflections
* Animated hour, minute, and second hands
* Digital time display overlay
* Responsive window resizing
* Atmospheric environment with particle effects
* Keyboard-based navigation and controls

## 🛠 Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* Three.js (r128)
* WebGL
* GLSL Shaders

## 🎮 Controls

| Key        | Action               |
| ---------- | -------------------- |
| ← →        | Rotate Camera        |
| ↑ ↓        | Adjust Camera Height |
| + / -      | Zoom In / Out        |
| Mouse Move | Rotate Light Source  |
| R          | Reset Camera         |
| L          | Change Light Color   |

1. Clone the repository:

```bash

```

2. Open the project folder.

3. Run `analog-watch-final.html` in a modern web browser.

No additional installation is required.
